📖 기능
+ Spring Boot Security 와 JWT 를 이용한 로그인 기능
+ Spring Data JPA 와 Oracle DB 를 이용한 회원 CURD 기능
+ Mail 라이브러리와 Redis 를 이용한 이메일 인증 기능
+ Validation 라이브러리를 이용한 유효성 검사 기능
+ REST API 기능
+ AOP 를 이용한 중복체크 로직 기능
+ Interceptor 를 이용한 로그 기능
